# Aftermarket Car Parts Store

Full-stack e-commerce site for aftermarket automotive parts.

## Tech Stack
- Frontend: HTML / CSS / JS
- Backend: Node.js + Express
- Payments: Stripe Checkout
- Hosting: Netlify (frontend), Railway or Render (backend)

## Features
- Shopping cart
- Stripe checkout
- Inventory tracking
- Order confirmation emails
- Admin dashboard
- Vehicle fitment selector
- Invoices & packing slips
